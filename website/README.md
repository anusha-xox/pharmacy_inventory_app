# Chandra Hi-Tech ENT Hospital Website

Professional static website for Chandra Hi-Tech ENT Hospital, Jhansi, created as a patient-facing hospital website for Dr. Anchal Jain.

## What changed
- Removed hallucinated statistics, fake testimonials, fake emergency claims, fake email IDs and unsupported departments.
- Used a doctor-led, appointment-first hospital layout.
- Used Inter and Playfair Display styling consistent with the earlier website direction.
- Replaced emoji-style icons with Font Awesome medical icons.
- Included actual public listing images already available in the project package.

## Files
- index.html
- services.html
- doctor.html
- contact.html
- sources.html
- styles.css
- script.js
- assets/images/hospital-exterior.webp
- assets/images/dr-anchal-jain.jpg

## Important production note
Public listing images have been included for the prototype because they are publicly visible online / already present in the working package. For production, confirm image usage rights or replace them with hospital-owned photos.

## Run locally
Open `index.html` directly or run:

```bash
python3 -m http.server 8000
```

Then open http://localhost:8000.
